🌐 Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and experience as a developer. It includes smooth animations, a modern UI, and a working contact form.

🚀 Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🎬 Framer Motion
- 📧 EmailJS

✨ Features

- Responsive and modern UI
- Smooth animations using Framer Motion
- Fully functional Contact Form (EmailJS integration)
- Projects showcase section
- About Me section
- Clean and minimal design

Direct Link of Portfolio:- https://rohitgupta-portfolio1.netlify.app/
